# 查词划句

## 提供的功能
- 单词双击选中自动弹出释义
- 可将单词添加到欧路的生词本
- 按 ctrl 划句, 通过百度 api 翻译
- 只要是 chominium 内核的浏览器，都可以使用本扩展。
